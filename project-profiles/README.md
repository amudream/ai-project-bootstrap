# Project Profiles

A project profile adapts the universal AI kernel to one business domain. Copy the relevant profile into `PROJECT_PROFILE.md` in a new project, or tell the agent to load it for the current task.

Use one primary profile at a time. Add secondary profiles only when the task truly spans domains.

Available profiles:

- `generic-ai-project.md`
- `alibaba-international-b2b.md`
- `customer-support-crm.md`
- `stock-review.md`
- `independent-site-outreach.md`
- `x-content-ops.md`
