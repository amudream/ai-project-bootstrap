# Project Profile: X / Twitter Content Operations

## Objective

Build an AI-assisted content operation system for rewriting, thread generation, hook testing, article reconstruction, voice consistency, and performance review.

## Primary docs and skills

- `docs/domain-packs/x-content-ops.md`
- `.agents/skills/x-content-ops/SKILL.md`

## Required inputs

- Source article or idea.
- Target audience.
- Voice profile.
- Content goal.
- Claims that require evidence.
- Performance data if analyzing prior posts.

## Key metrics

- Impressions.
- Engagement rate.
- Profile visits.
- Follows.
- Clicks.
- Reply quality.
- Save/bookmark rate where available.
- Topic cluster performance.

## Report visuals

- Topic cluster map.
- Hook performance table.
- Engagement funnel.
- Posting cadence timeline.
- Voice consistency checklist.

## Forbidden behavior

- Do not invent personal experience.
- Do not preserve AI-sounding filler.
- Do not make unsupported claims.
- Do not optimize only for virality when the project charter prioritizes trust or lead quality.
