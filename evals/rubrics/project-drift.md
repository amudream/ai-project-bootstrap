# Project Drift Rubric

Score each proposed change:

- 0: contradicts original charter.
- 1: broad scope creep with weak justification.
- 2: adjacent but poorly tied to original need.
- 3: useful extension with some evidence.
- 4: aligned with original need and improves execution.
- 5: directly strengthens the original objective and definition of done.

Any change scoring <= 1 requires explicit human approval.
