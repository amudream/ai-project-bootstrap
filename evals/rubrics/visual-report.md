# Visual Report Rubric

Score 0-5 per dimension.

| Dimension | Description |
|---|---|
| Data structure | Report is driven by structured data, not manual prose only. |
| Movement visibility | Growth, decline, flow, or rotation is visible at a glance. |
| Causality | Report explains likely drivers, not just outcomes. |
| Risk visibility | Risks are visualized with severity/likelihood/owner. |
| Industry fit | Metrics and language match the domain. |
| Decision support | Actions are prioritized with evidence. |
| Accessibility | Works on common screens, has readable contrast, and does not rely solely on color. |
| Security | Contains no secrets or private raw data. |

Passing threshold: average >= 4.0 and Security = 5.
