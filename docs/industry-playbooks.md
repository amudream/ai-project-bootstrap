# Industry Playbooks

Use these lenses when adapting the generic framework to specific industries.

## Universal project anatomy

Every project needs:

- Charter: why it exists.
- Stakeholders: who cares.
- Inputs: data, files, tools, constraints.
- Outputs: product, report, decision, automation, code.
- Metrics: how success is measured.
- Risks: what can fail.
- Loop: how it improves after launch.

## SaaS / software product

Ask the agent to inspect:

- Activation path.
- Retention loop.
- Pricing and packaging.
- Support burden.
- Observability.
- Experimentation plan.
- Roadmap dependencies.
- Technical debt and test coverage.

Best report visuals:

- Activation funnel.
- Retention cohorts.
- MRR movement waterfall.
- Churn reason heatmap.
