# Eval Designer Skill

## Procedure

1. Define what success means.
2. Split objective into measurable dimensions.
3. Prefer deterministic tests where possible.
4. Add rubric checks for qualities that require judgment.
5. Define thresholds and stop conditions.
6. Create representative cases.
7. Ensure outputs are machine-readable.
8. Log baseline and improvements.

## Eval types

- Exact match.
- Schema validity.
- Unit/integration test.
- Secret scan.
- Citation coverage.
- Visual report rubric.
- Anti-drift rubric.
- Human review checklist.
