# Domain Report Blueprints

Use these blueprints to make HTML reports more useful than markdown. Each report should be generated from structured data and should preserve data provenance.

## Alibaba International Station report

Required sections:

1. Executive summary.
2. Funnel: exposure → click → visitor → inquiry → qualified inquiry.
3. Product portfolio matrix.
4. Keyword opportunity matrix.
5. Country heatmap.
6. Category traffic trend.
7. Inquiry quality breakdown.
8. Recommended actions with validation metrics.

Recommended visuals:

- Time-series line chart for exposure, clicks, visitors, inquiries, and qualified inquiries.
- Funnel chart.
- Country heatmap.
- Scatter plot: traffic versus conversion.
- Waterfall for ad spend to qualified inquiries.

## Stock review report

Required sections:

1. Market regime.
2. Sector / theme rotation.
3. Candidate watchlist.
4. Annotated setups.
5. Trigger and invalidation table.
6. False-positive review.
7. Backtest/replay result if available.
8. Risk summary.

Recommended visuals:

- Candlestick charts with annotations.
- Volume and turnover panels.
- Sector heatmap.
- Limit-up timeline.
- Drawdown curve.
- Score distribution.

## Independent site outreach report

Required sections:

1. Market and prospect summary.
2. Prospect fit ranking.
3. Product matching.
4. Outreach angle library.
5. Email sequence.
6. CRM export table.
7. Compliance and deliverability notes.

Recommended visuals:

- Prospect fit matrix.
- Outreach funnel.
- Country/category map.
- Objection heatmap.

## X content report

Required sections:

1. Content thesis.
2. Audience lens.
3. Hook variants.
4. Final posts / threads.
5. Voice consistency notes.
6. Claim risk check.
7. Performance review if data exists.

Recommended visuals:

- Topic cluster map.
- Hook comparison table.
- Posting cadence timeline.
- Engagement funnel.

## CRM/support report

Required sections:

1. Demand summary.
2. Inquiry quality funnel.
3. Objection heatmap.
4. Product request trends.
5. Response quality issues.
6. Escalations.
7. Sales/product/documentation actions.

Recommended visuals:

- Demand cluster map.
- Objection heatmap.
- Response-time trend.
- Stage movement funnel.
